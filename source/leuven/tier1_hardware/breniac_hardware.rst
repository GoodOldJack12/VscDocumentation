Breniac hardware
================

Breniac is the VSC's Tier-1 cluster.

- 408 skylake nodes

    - 2 Xeon Gold 6132 CPU\@2.6GHz, 14 cores each
    - 192 GB RAM
    - 75 GB SSD local disk

- 436 broadwell nodes

   - 2 Xeon E5-2680v4 CPUs@2.4GHz, 14 cores each
   - 128 GB RAM
   - 75 GB SSD local disk

- 144 broadwell nodes

   - 2 Xeon E5-2680v4 CPUs@2.4GHz, 14 cores each
   - 256 GB RAM
   - 75 GB SSD local disk

The nodes are connected using an Infiniband EDR network.
